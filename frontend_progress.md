## Day 1 â€” Detailed Plan (Tech Stack + Setup)

### Tech Stack (Decision + Rationale)
- **React + Vite**: fast dev server and build performance.
- **TypeScript**: safer UI contracts with backend APIs.
- **Tailwind CSS + Headless UI**: quick layout for 3-column dashboard.
- **React Router**: page routing and session flow.
- **TanStack Query**: API fetching and caching.
- **Zustand**: lightweight session/module UI state.
- **Recharts or ECharts**: charts for performance dashboard.

### Day 1 Tasks (Detailed)
1. **Initialize Frontend App**
   - `npm create vite@latest` with React + TS.
   - Add base folder structure: `src/pages`, `src/components`, `src/services`, `src/store`.
2. **Install Core Dependencies**
   - `react-router-dom`, `@tanstack/react-query`, `zustand`.
   - Tailwind + PostCSS config.
3. **Create App Shell Layout**
   - 3-column grid: Left Nav, Center Content, Right AI Bot.
   - Placeholder components for each column.
4. **Routing Skeleton**
   - `/login`, `/session/start`, `/session/upload`, `/dashboard`, `/performance`.
5. **Design Tokens + Base Styling**
   - Define typography and spacing.
6. **Daily Notes**
   - Add progress notes and blockers.
